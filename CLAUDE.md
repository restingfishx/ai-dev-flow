
# 项目描述

这是一个基于claude code，搭建的一套AI软件开发体系，主要特点有：
1. 角色分工明确。目的是让每个角色职责与技能更加专业，从而具有更规范、更高质量的产出
2. 精细任务拆分。目的是实现对任务的细粒度追踪，便于对某个任务进行反复打磨（特别是开发编码阶段）。
3. 流程全程可控。有审核、反馈机制，使用户掌握流程推进的主动权。
4. 基于claude code现有能力实现。

# 工作目录

project/

## 说明
project目录下是完整的clude code工程结构，当实际项目需要利用这套体系进行开发时，只需通过 `sync-claude.sh`命令将project/下的文件拷贝到实际项目根目录中即可。
